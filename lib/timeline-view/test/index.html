Timeline

A vertical timeline view. Simply pass it a collection of events and you'll retrieve a structurally styled timeline with icons from ionicons.

API

First create some events. The type-attribute should be compatible with ionicons icon-naming and description should be in the markdown-syntax.

var event1 = new Event({
  'timestamp': new Date,
  'type': 'beer',
  'description': '# party'
});
var events = new Collection([event1]);

Then initialize your timeline:

var timeline = Timeline(events);
document.body.appendChild(timeline.el);
